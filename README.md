# Documentation

# Quick use

``npx git-badges ``

# Installation

``npm i git-badges -g ``

# Usage

``git-badges <port number> mongodburl``

# Example
``git-badges 3000 mongodb://localhost:27017/ghviews ``

# Env

## Creating Env
``touch .env``

# Env Example

## Usage
```
MONGO_URL=<url>
PORT=<Port>
```
## Example
```
MONGO_URL=<url>
PORT=3000
```
